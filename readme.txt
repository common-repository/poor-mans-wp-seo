Packs all the important features to control SEO on your site. This is a free alternative to plugins such as All In One SEO Pack and Yoast SEO.

== NOTICE ==
This plugin is no longer developed. Something better has emerged. Please check out [Open WordPress SEO](https://wordpress.org/plugins/open-wp-seo/).

== Description ==

Command how you show up in search results! With Poor Man's WordPress SEO search engines find your content more easily, index your content faster, and give your site better ranking.

== Features ==

Poor Man's WordPress SEO is all-in-one solution for WordPress SEO needs. The plugin adds a status screen to your dashboard to show if there is any action you need to take to improve SEO.

Post editor will have a new section to fine-tune your post's meta tags. The section will also display suggestions on how to optimize the content your are creating.

Routine actions such as creating and updating a sitemap and notifying search-engines about new content are taken care of automatically.

NOTICE: Poor Man's WordPress SEO is no longer being developed. A better, 100% free, open-source alternative has been built on the bases of this plugin. Please check out [Open WordPress SEO](https://wordpress.org/plugins/open-wp-seo/).
